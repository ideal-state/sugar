### 如何使用

#### Maven
```xml
<!--pom.xml-->
<dependency>
    <groupId>team.idealstate.sugar</groupId>
    <artifactId>sugar</artifactId>
    <version>${version}</version>
</dependency>
```

#### Gradle
```groovy
// build.gradle
dependencies {
    implementation "team.idealstate.sugar:sugar:${version}"
}
```
```kotlin
// build.gradle.kts
dependencies {
    implementation("team.idealstate.sugar:sugar:${version}")
}
```

### 如何构建

```shell
# 1. 克隆项目到本地
git clone https://github.com/ideal-state/sugar.git
# 2. 进入项目根目录
cd ./sugar
# 3. 构建项目
./gradlew jar
```
